Техническое задание на проект: Контекстный Генератор Примеров

1. Цель проекта:
Сервис помогает изучающим английский язык находить контекстные примеры употребления слов и фраз в реальной речи. Вместо сухого перевода пользователь получает определения с разбором по частям речи, живые примеры из словарей и параллельных корпусов, а также их автоматический перевод на русский язык. Сервис агрегирует данные из нескольких внешних API в едином интерфейсе.

2. Роли пользователей:
- Гость (неавторизованный пользователь): ввод слова или фразы, просмотр определений и примеров, история запросов (сохраняется в браузере), переключение темы оформления.
- Администратор: управление контентом через Django Admin (просмотр истории поисковых запросов, управление кэшем определений и примеров).

3. Модели данных:
- Модель 1: SearchQuery (Поисковый запрос) — query_text (текст запроса), ip_address (IP пользователя), user_agent (информация о браузере), created_at (дата и время), results_count (количество найденных примеров).
- Модель 2: WordDefinition (Определение слова) — word (слово, уникальное), definition_data (структурированные данные определения в JSON), translation (перевод на русский), transcription (транскрипция), created_at, updated_at.
- Модель 3: Example (Пример употребления) — word (связь с WordDefinition), text (текст примера на английском), translated_text (перевод на русский), source (источник: yandex/dictionary/linguee), score (числовой рейтинг), rating (текстовая оценка), word_count (количество слов), created_at.

4. Ключевой функционал:
- Пользователь вводит слово или фразу в поле поиска, система отправляет запрос к внешним API, агрегирует определения и примеры, ранжирует их по релевантности и выводит результат с переводом.
- Пользователь нажимает на любой запрос из истории, и система автоматически выполняет поиск без повторного ввода.
- Пользователь переключает тему оформления (светлая/тёмная), и выбранная тема сохраняется в браузере.

5. Внешние интеграции / Аналитика:
- Yandex Dictionary API — получение определений, транскрипции, синонимов и примеров употребления.
- Dictionary API (dictionaryapi.dev) — резервный источник определений.
- Linguee (парсинг HTML через BeautifulSoup) — получение контекстных примеров из параллельных корпусов.
- MyMemory Translation API — автоматический перевод примеров с английского на русский.
